# brush_size_docker
Compact brush size docker for Krita

This is adapted from the "Quick Settings Docker" plugin that ships with Krita. But it only includes brush sizes and is intentionally more compact.

You can assign shortcut keys to increase and decrease the brush size according to the docker sizes.

When Krita's brush size is at a value that does not exactly match (e.g between 30px and 40px), the smallest closest match will be highlighted.
